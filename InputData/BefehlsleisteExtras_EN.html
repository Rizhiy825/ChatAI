Home bar Extras

Tools

Click the button to call up the tool profiles that are read into SmartWOP.
In the drop-down menu, select the tool you want to display in the profile.
You can create or modify tools in the tool profile window.

Link

Select the workpieces you want to link and press the button.
You can only link workpieces if they are in the same volume.
Drawer fronts cannot be linked.

Unlink

The button only becomes active when you select a workpiece that you can unlink.
You can only detach workpieces if they have already been linked.
Select the workpiece to be unlinked and click the button to unlink the linked panel.

Component Save as

Select a volume and click the button to save the construction in this volume as a variable component.
You can only save one volume at a time, including its contents, as a component.
A saved component retains all the machining, fittings and materials that were inserted.
A component is a variable construction that adapts to the selected volume.
Saved components can be pinned and selected as quick access in the gallery.

Disassemble Component

When you drag a component into your design, you can explode the component to add or change machining.
Select the component to be disassembled and click the button.
The component is exploded and the items can be selected individually.

Machining Collisions

Activate the button to check the project for collisions of the machining.
If there are collisions, they are marked in red.

Show Material Removal

Activate the button to make the material removal of individual or several items visible.
If you have activated the button, the material removal is applied to the items you select.

Numbering

Enter the start number of the component numbering.
Specify the step of consecutive component numbering.

Explosion

Click the button to get an explosion view.
In the explosion view, items are displayed numbered.
Indicate the distances of the items in the explosion in X/Y/Z.
Move the fronts in the explosion in X/Z to the desired dimension.
Activate the Show Connections button to display a line between the connected items in the explosion.

Select and Move

Select and move is the control mode automatically used by SmartWOP.
This button is active until you activate another button, such as 'Rotation'.

Additive Selection

Activate the button to select several items or volumes.
You have the option of using the same function by left-clicking with the Ctrl key.

Subtractive Selection

Activate the button to deselect items or volumes.
You have the option of applying the same function by Alt left-clicking.

Frame

Select the button to select several or all items at once.
Only items that are completely within the frame are selected.

Section

Activate the button to specifically select the entire volume.

Invert Selection

Select the button to invert the selection of the selected elements.

Rotation

You can rotate bases together with their contents.
Click on the arrow below the button.
Select the number of degrees by which you want to rotate the volume.
Activate the button and select the volume to be rotated.
You can rotate the volume around the desired axis by holding down the mouse button.
Clicking the button again deactivates the rotation.

Hide Fittings

Activate the button to hide all fittings in the construction.
The machining of the hidden fittings remains visible.
