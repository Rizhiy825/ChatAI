Fittings

Add Fittings

To insert fittings, you need item to which you can assign the fittings.
You can drag and drop fittings onto the items.

Manufacturer and Supplier

You have the option of specifying the manufacturer and supplier of the fitting.
To find fittings for which you have specified manufacturers and suppliers more quickly in the fittings database, you can select the manufacturers and suppliers to the left of the fittings to display only these in the list.
The selected manufacturers and suppliers are displayed with a blue background.
To select or deselect individual manufacturers or suppliers, left-click on the manufacturer or supplier.

Custom Handle Profile

You have the option of defining handle profiles yourself and then importing them.
To define a profile yourself, you must first create a .txt file, e.g. using Microsoft's editor.
When the file is created, you must define the contour of the profile using X and Y in the 2-dimentional.
You always determine the contour line via the contour points (X= and Y=).
The contour must first be given a starting point, e.g.(X=0 Y=0).
The contour line is always drawn directly from contour point to contour point.
The entire contour must be self-contained again at the end.
The length of the handle is not specified in the file, as it automatically refers to the entire length of the item.

Duplicate Fitting

In the drop-down menu of the top fitting selection bar, you can duplicate fittings and customize them as desired.
Open the drop-down menu by clicking on the arrow of the upper fitting selection bar.
Select the fitting you want to duplicate and click the second button of the drop-down menu.
Duplicating a fitting opens the edit mode.
Adjust the fitting and save the changes of the duplicate by confirming the changes via the orange arrow of the dropdown menu.
Duplicate fittings can help you to quickly adjust a fitting in this way without having to create a new fitting.

Rename fitting

In the drop-down menu of the upper fitting selection bar, you can rename fittings.
Select a self-defined or duplicated fitting and activate the fitting editing mode.
Now click on the third button of the drop-down menu to rename the selected fitting.
You can rename fittings only if you have created or duplicated on your own.

Set Favorites

You have the possibility to mark selected fittings as favorites so that you can quickly find them in your fittings.
To do this, select the fitting you want to set as a favorite and click on the small blue bordered star in the icon of the fitting.
To remove a fitting as a favorite, click the favorite star again.
SmartWOP will ask you if the fitting should be removed from the favorites list.
Activate the orange favorites star under the hardware selection bar to display only your favorites in the hardware list.
Fittings that have already been combined in a variable database, such as the Blum variable pot hinges, cannot be marked as a favorite.

Fitting Search bar

In the fitting search bar, you can filter out and display individual fittings from your entire fitting database using the name of a fitting or the article number of a fitting.

Blum variable Hinges

To add a hinge to your construction, you need a door that you can put the hinges on.
Blum's combined hinge fittings lists provide you with flexible and quick choices to match the hinge to your construction.
The icons show you for which overlay situation, the individual data banks can be used.
The different applications are divided into standard application, wide angle hinges and profile door application, each for inside and outside hinges.
When using the variable Blum hinges data base, the pot hole spacing is taken into account and automatically adjusted when the fittings are pulled in, depending on the joint dimensions.

Count: Specify the number of pot bands.
Step (max.): Specify a dimension at which distances the pot bands should be located at maximum.
You have the option of either specifying the number of pot bands or setting a maximum step at which the pot bands should be spaced.
To do this, click on the icon on the far right in the line.
Top Distance: Specify the desired distance of the upper pot band.
Bottom Distance: Specify the desired distance of the lower pot band.
Positions: In the dropdown menu you have the possibility to determine the positions of the individual pot bands.
Configuration UI: In the UI configurator, you have the option of specifying an individual name for the fitting in the CSV material list.
Call the name Template Editor via the pencil icon on the right side of the bar.
Select the information stored in the template editor with a double left mouse click to add the selected information to the fitting in the CSV material list.
Configuration: In the drop-down menu, you have the option of specifying settings for the fitting and inserting the cup hinge and the mounting plate directly via the filter function to suit the fitting.
To activate the configurator, click on the filter icon in the "Hinge" line.
If the configuration has been set in such a way that there is no possible combination for the cup hinge or mounting plate, the hinge or mounting plate is not output when the fitting is pulled in..
The fitting information entered in the configurator is processed directly by SmartWOP and the matching cup hinge and mounting plate are selected directly in the "Accessories" field and used directly when inserting the fitting.
Accessories: In the drop-down menu you have the possibility to select the hinge and mounting plate of the fitting.
Hinge: Use the drop-down selection on the right of the line to select the hinge.
Activate the filter function to consider the information set in the configurator for the selection of the hinge and mounting plate.
Deactivate the filter function to specify the selection of the hinge and mounting plate itself.
Mounting Plate: Use the drop-down selection on the right of the line to select the mounting plate.
Alignment: Select on which side the hinge is to be attached.
Bind outer to Shelf Bores: Tick the box to orient the upper and lower hinge, for mounting plate machining, directly to the system hole..
The outer hinges are set on the next possible system hole line machining, the positions are assigned to the fitting.
Bound inner to Shelf Bores: Tick the box to let the middle hinges, for the mounting plate machining, orient directly to the system hole line.
The middle hinges are set on the next possible system hole line machining, the positions are aligned to the fitting.

Insert Angle bracket

To insert an angle fitting into your construction, you need an item to which you can attach the angle fitting.
Select the desired angle fitting and drag it into your construction.
In the template window you have the possibility to make adjustments to the fitting.

Supplier: Specify the supplier of the fitting.
Select an already created supplier via the dropdown selection.
Delete the supplier entered in the line from your list by clicking on the X symbol in the line.
Article: Specify the article number of the fitting.
Count: Specify the number of angle fittings required.
Step (max.): Specify a dimension at which distances the angle fittings should be at maximum.
You have the possibility to specify either the number of angle brackets or a maximum step at which the angle brackets should be spaced.
Top Distance: Enter the distance of the upper angle bracket from the outer edge of the item to the angle fitting.
Bottom Distance: Enter the distance of the lower angle bracket from the outer edge of the item to the angle fitting.
On the right side of the line you have the possibility to lock the distances of your angle fitting by clicking on the lock symbol, in order to always set the distances of the angle fitting to a fixed value of top/bottom, even when saving the construction as a component.
Positions: Specify the positions of the respective angle brackets.
Distance to Edge: Specify the distance of the fitting from the edge of the item.
Consider the distance to the edge of the workpiece when mounting an angle fitting e.g. on a back panel that is grooved in other workpieces.
Edge: In the drop-down menu you have the option to select the reference edge for the angle fitting.
Keep in mind that depending on the reference edge, the distances may not be correct and the angle fitting may therefore be outside the volume.
The Selected reference edge determines in which direction the angle fitting runs and specifies the calculation points for the fitting.
Top Side/Bottom Side: Check the box if you want to change the processing side of the fitting.
Switch Arms:Set the check mark if you want to exchange the angle arms.
Mirror: Check the box if you want to mirror the fitting on the opposite side.
Additional Info: Specify additional information that you want to add to the fitting on the material list.

Insert Lock Systems

To add a lock system to your construction, you need a door on which you can mount the fitting.
The locking system is available for both the right and left hinge side.
Select the lock system fitting and drag it into your construction.
In the template window you have the possibility to make adjustments to the fitting.

Supplier: Specify the supplier of the fitting.
Select an already created supplier via the dropdown selection.
Delete the supplier entered in the line from your list by clicking on the X symbol in the line.
Article: Specify the article number of the fitting.
Horizontal Distance: Specify the horizontal distance of the lock system.
The distance of the locking system is taken from the center of the lock pot hole.
Vertical Distance: Specify the vertical distance of the lock system.
The distance of the locking system is taken from the center of the lock pot hole.
Vertical Align: Use the drop-down selection to choose the alignment of the vertical spacing.
Absolute Position: SmartWOP shows you on which position the lock system is from the volume bottom edge.
The position is indicated from the center of the lock hole.
Minimal Distance for Bar Guide: Enter a minimum distance at which the bar guide should be located.
The distances of the bar guide are calculated automatically.
Vertical Pocket: In the drop-down menu you have the option to make adjustments for the lock passage.
Radius Gap: Specify whether and how much the radius to be machined for the lock opening should become larger.
Tool Number: Specify the tool number for machining.
Select the required tool via the drop-down selection, on the left in the line.
Tool Feedrate: Output the feed rate for machining.
Fitting:In the drop-down menu, SmartWOP shows you which parts are required for the retracted lock system.
Additional Info: Specify additional information that you want to add to the fitting on the material list.
