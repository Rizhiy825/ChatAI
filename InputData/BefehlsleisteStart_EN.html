New

Start a new SmartWOP project, in an empty scene..

Office Mode

In planning mode, the items are renumbered throughout.
Care should be taken not to export the items until the construction is complete.
The numbering of all items may change if you add or change items after export.
SmartWOP will ask if you want to go into workshop mode when you export all items to the machine.
Projects that you save in office mode are also opened in office mode.

Workshop Mode

In workshop mode, the numbering of the items is frozen.
Care should be taken not to make any changes to the design if the items have already been exported.
If you add or change items in production mode, only the items that have received changes are renumbered.
If you want to go back to office mode, SmartWOP will notify you that all item numbers will be recalculated.
Projects that you save in workshop mode are also recalled in workshop mode.

Delete

Delete the selected Items.

Duplicate

You can duplicate bases together with their added items.
Select an item that lies in the volume to be duplicated or select the entire volume to be duplicated.
Click on the Duplicate button.
It is not possible to duplicate individual items.

Mirror

You can mirror bases together with their added items.
Select an item that lies in the volume to be mirrored or select the entire volume to be mirrored.
Click on the arrow below the item Duplicate, there you will find the button to mirror the volume.
It is not possible to mirror individual items.

Allow Overlapping

If you activate the button Allow Overlapping, you have the possibility to drag bases into each other.
Select the volume to be moved and move the volume to the desired position using the arrow.
This function can be useful if you have several volumes in your scene that you want to move.
By clicking the button again, overlaps are deactivated again.
Volumes that still overlap can be pushed apart again without activating the button.

Rotation

You can rotate bases together with their added items.
Click on the arrow under the button.
Select the degree by which you want to rotate the volume.
Activate the button and select the volume to be rotated.
You can rotate the volume around the desired axis by holding down the mouse button.
Clicking the button again deactivates the rotation.

Volume

The button is used to select individual volumes.
Press the button again to be able to select individual items.

Activate Volume

If you want to work in a specific volume, you can use this button to restrict the editing to the activated volume.
Use the 'Volume' button to select the volume to be edit and activate it with the button.
The activated volume is now displayed with a green outline.
SmartWOP does not allow any operations outside the activated volume.
To release the editing for other volumes, click the button again.

Select and Move

Select and move is the control mode automatically used by SmartWOP.
This button is active until you activate another button, such as Rotation.

Additive Selection

Activate the button to select several items or volumes.
You have the option of using the same function by left-clicking with the Ctrl key.

Subtractive Selection

Activate the button to deselect items or volumes.
You have the option of applying the same function by Alt left-clicking.

Frame

Select the button to select several or all items at once.
Only items that are completely within the frame are selected.

Revert Selection

Select the button to invert the selection of the selected elements.

Machining

Select the button to switch to edit mode.
Machine operations can be added to items in the machining mode.

Hide Fronts

Select the button to hide all fronts.
If you hide the fronts, you have the option of selecting interior items more easily.
Before exporting your construction, think about that you are showing all the items.
Only the items that are in the scene are taken into account when exporting.

Hide Selection

Select the button to hide selected items.
If you hide individual items, you have the option of better selecting internal items.
Before exporting your construction, remember to display all items..
Only the items that are in the scene are taken into account when exporting.

Hide Unselected

Select the button to display selected items alone.
If you show individual items, you have the option of machining or exporting these items separately.
Before exporting your construction, think that you are showing all the items.
Only the items that are in the scene are taken into account when exporting.

Show All

Select the button to display all items.
Before exporting your design, think that you are viewing all the items.
Only the items that are in the scene are taken into account when exporting.

Show Separators

Select the button to show or hide all separators.
If you hide all separators, you have the possibility to better select internal items.
To show the separators again, click the button again.

Solid mode

Select the button to move from Transparency mode to Solid mode.
In the submenu of the button, we have the option of switching contour trains on and off in material mode.
To return to transparency mode, click the button again.

activate snapping

Activate the button to set a specific minimum step size for moving objects.
Enter this minimum distance in mm in the field below.

Magnet

Activate the magnet to allow the automatically generated magnet points when moving.
Enter in the field below the magnet from which area the workpiece is to be added to a magnet location.

Measure

The tape measure allows you to measure distances between the selected points.
Activate the button to activate the measuring tape.
Select a starting point and drag a dimension line to the desired measuring point.
If the measuring points are on different axes, the measurement is additionally output in X/Y/Z.
The display of the measurements is temporary and is automatically hidden when a starting point is selected again.

Dimensioning

The dimensioning allows permanent dimensioning of the distances.
Click on the button to activate the dimensioning.
Select a starting point and drag a dimension line to the desired measuring point.
You have the option of moving the dimension display by selecting a dimension and dragging it to the desired position with the mouse button held down.
Dimensions are permanent and can only be deleted using the delete function (Del)..
In the View-task, you have the option of showing and hiding dimensions.

Comment

Activate the button to add a comment to specific items or volumes.
Click on an item to add an individual comment.
To identify which item you have selected, the comment is given as the item name.
Enter the desired comment in the input window.
You have the option of moving the comment display by selecting a comment and dragging it to the desired position while holding down the mouse button.
Comments are permanent and can only be deleted using the delete function (Del).
In the View-task, you have the option to show or hide comments.

Collision

Activate the button to check the project for collisions of the machining elements.
The corresponding points are marked in red.

Edit Top

Select an item and activate the button to insert machining via WoodWOP, on the top side.
The side to be machined is marked with a yellow arrow.

Edit Bottom

Select an item and activate the button to insert machining via WoodWOP, on the Bottom Side.
The side to be machined is marked with a yellow arrow.

Activate Assistant

Activate the button to activate your assistant.
In the image at the bottom right, under the menu item Assistants, you can create an assistant.
The assistant can be taught to speed up and simplify re-occurring constructions.

Export Tuning

Activate the button to open the Export Tuning.
In the Export Tuning submenu we have the option of displaying or hiding a virtual machine table.
Select an item that you want to machine using Export Tuning or export to the WOP environment.
In the template window you will find further options for optimizing your workpiece.

Panel:Select whether the workpiece is to be made from an unformatted raw plate or an already formatted prefabricated plate.
Formatting Type: Select how to format.
Machinings Before Formatting: Check the box if formatting is to be done after machining.
Non-Rectangular Cuts After Formatting: Place a tick if the diagonal cuts or chamfering is to be made after the edge processing.
Stage Order: Select whether the workpiece should receive the edge before or after machining.
Sides Order: Specify whether the outer/inner side of the workpiece is to be machined first.
Merge Export Programs: Specify how the programs of the editing sides are to be exported.
Using Mirrors: Select whether the workpiece to be machined on both sides is to be machined on the mirrored workplace in the second side machining.
Use NC Stop: Select whether the workpiece to be machined on both sides is to be machined by NC stop in the second side machining in the same program.
Two Plates, Using Mirrors: Dont´t Merge: Select whether the workpiece to be machined on both sides is to be machined in the second side machining in a separate programs.
Auto PLace: Place a tick if the workpiece is to be positioned as in the option setting. Remove the tick if you want to change the positioning manually.
Use retractable stop bold: Set a check mark if you use fold-out fences for machining the workpiece.
Formatting Prod. Part Offset: Specify the desired production part offset dimension, of the workpiece when formatting on the machine.
Stage 1: SmartWOP shows you how and in which order the processing steps are carried out.
Tool Numbers: Specify a desired tool number for machining for the selected workpiece. If you do not make any changes, the tools from the option setting are used.
Profile: Change settings in your WOP environment by duplicating the design and making changes in the duplicate.
It is important that you create a duplicate, as the pattern is reset to default after each update to avoid potential errors.
Machine Set-UP Configuration: Select on which machine the workpiece is to be machined.

Center View

Click the button to center the view on the selected item or volume.
