Layout Bar

Alignment

The Alignment indicates how directions are calculated in SmartWOP.

Ground

Select a volume and click the button to bring the volume and its contents to the baseline.

Select and Move

The button is active by default to select and move workpieces and volumes.

Additive Selection

Select the '+' button to select multiple items or volumes.
You have the option of using the same function by left-clicking with the Ctrl key.

Subtractive Selection

Select the '-' button to deselect items or volumes.
You have the option of applying the same function by Alt left-clicking.

Frame

Select the 'Frame' button to select several or all items at once.
Only items that are completely within the frame are selected.

Section

Activate the button to select all workpieces that are within two separators.

Invert

Select the 'Invert' button to select all unselected items.

Wireframe

Activate the button to display the construction in polygon mode.

Screenshot

Click the button to make a screenshot.

Presentation

Activate the button to switch to presentation mode.
Use the mouse position to determine the rotation speed of the presentation.
Click a button on the mouse to exit presentation mode.
