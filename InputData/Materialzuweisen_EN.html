Assign materials

Materials selection window

As soon as you switch to material mode during a construction, the selection window opens.
You can use the selection window to decide whether surfaces are to be provided with a new material and which edging is to be applied to the item.
Select one or more items in your design to expand the selection window.
Items: Tick the box to make all items used in the design, except for drawer insides and components, selectable for the material selection.
Components: Tick the box to make the components used in the design selectable for the material selection.
Drawer Insides: Tick the box to make the internal parts of the drawers used in the design selectable for the material selection.

Material template

You can use the material template to select different panel materials and edge materials and add them to your items.
Name: SmartWOP shows you the material name of the selected material.
Panels: Select the materialpanel you want to add to your selected items.
Grain Rotation Angle: Enter the grain angle to rotate the grain on the selected items.
Edges: Select the edge material that you want to add to your selected items.
Edges banding: Select how the edges are to be arranged for the selected items.
By default, the front edge goes through and the rear edge jumps in between the side edgings.

Edge assignment via selection window

If you have selected one or more items in material mode and the selection window has expanded, you can select the edging.
Select the material you want to add to your selected items from the material database.
Use the selection window to choose which edges are to be known for the selected items.
Items:By clicking on the box next to the position, you can select the edging for the right, left, front, back, top, bottom or other edge.
Arrangement:In the column below the arrangement, the order in which edges are to be approached is displayed.
The sequence of the edge arrangement is determined by which edges are set to be continuous or recessed via the material window.
Materials: Use the material template to select the desired edge material to be added for the selected items and edges.
Confirm the assignment of the edging using the orange tick in the material.

Material assignment via selection window

If you have selected one or more items in material mode, you have the option of applying a new material to the surfaces of the selected items.
Select the material you want to add to your selected items from the material database.
Select the surfaces via the selection window to change the material.
Panels Select the materialpanel you want to add to your selected items.
If the item with a certain thickness is used in the construction, a panel material with a different thickness is not automatically added.
Confirm the assignment of the material using the orange tick in the material.
