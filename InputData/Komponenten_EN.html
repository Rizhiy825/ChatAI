Components

Preface

SmartWOP offers the option of saving variable components in order to use all the constructions that have been planned within a volume in new constructions on an ongoing basis.

Function

Constructions that are required again and again can be created and archived in SmartWOP as variable construction components.
This makes it possible to use prefabricated systems variably in the construction of structures.
For example, you can use do-it-yourself drawer boxes.
To do this, your drawer variant is created once in SmartWOP and saved and stored as a variable component.
This component can now be arranged variably in volumes of any size.
The drawer construction automatically adapts to the assigned volume on all sides when it is pulled in.
The drawer does not have to be rebuilt again and again in every new construction.
SmartWOP components support machining (system holes), fittings such as hinges or drawer systems and materials.
This means that complete cabinet constructions can be created as components.
Many practical applications such as kitchen base units, wall units, tall units and wardrobe variants can be easily customised and used in a variety of ways.

Areas of application

SmartWOP components can only be created as rectangular constructions.
Constructions with diagonal items such as back angled or slanted bases are not considered as SmartWOP components.
Inserted components can be subsequently adapted and changed.
On the one hand, adjustments to the components are made via entries in the properties window.
On the other hand, a resolution of the components is possible, which can then be completely changed.

Create Components

Create a construction that is planned within a base volume.
The design can also be carried out by an assistant.
Once you have finished planning the design, you must now save it as a SmartWOP component so that it can be incorporated into your other designs on an ongoing basis.

Save components

If the created project is saved via "File" or the save as option is used, the design is saved as a project.
In that case, the construction is not variably supported.
SmartWOP projects are labelled with the file extension .swp. sw stands for SmartWOP and p stands for project.
There are two options available in SmartWOP for saving a design as a variable component.
You have the option of saving the component via "File" - "Component" - "Save as" or via "Extras" - "Component" - "Save as".
The path for the storage location can be set in the SmartWOP options under "Directories".
If a valid storage path is stored under Component, this is taken into account accordingly.

Gallery

Folders can be pinned in the gallery to gain quick access to the components you have created yourself and to use them in your constructions via drag and drop.
You can create folders with additional subfolders for different types of SmartWOP components.

Use components

A SmartWOP component can be inserted variably into a volume.
To do this, drag a basic volume onto the base area and customize its width, height and depth.
Then drag in the desired component via the Gallery category.
The component automatically adapts to the base body on all sides with all its contents.

Adjust components

Inserted components can be customized on all sides via the properties window.
To do this, change the height dimension of the component.
The component is adjusted upwards and downwards in the same proportion.
The changes are taken into account when specifying the distances (distance top, distance bottom).
A customized component can be moved directly using the direction arrows in the volume.
The changes are taken into account in the distances in the properties window.
The dimensional changes can also be made directly via the dimensions of the distances.
If a dimension is changed, the opposite dimension is automatically corrected.
Lock symbols can be used to lock the component in the respective direction in the event of a size change.
If, for example, the distance at the bottom is set to 0 and locked with the lock symbol, the excess dimension is deducted from the top when adjusting the height of the component.
The dimension can only be fixed in one direction.
The actual size of the component cannot be changed by locking.
A simultaneous locking of the top and bottom distances is not taken into account.
Notice: A component cannot be larger than the actual solid.
Changes to the size are only taken into account in reductions.

Assign material to components

If a component is called up by double-clicking, the material can also be changed individually.
Simply select items of the component and assign material.
Use the Frame function to select the entire construction and assign material.
Then save (overwrite) the component or save it with a new name as a new component.
Components that have already been assigned to a construction are not taken into account when changing the material of an entire project.
A component is to be seen as a closed system.
As long as a component is not disassembled, the material cannot be changed directly.
To change the material of an undissembled component in a project, the Add component option must be ticked in the selection window in material mode.
Only then can the material be assigned to a component or changed.

Disassemble component

A component can be disassembled in order to adapt or change items or the machining of an inserted component.
A component can only be disassembled if no scaling or position locking is provided in the properties window.
All distance values must be set to 0. All lock symbols must be open.
Select the component you want to disassemble.
If all distance values are set to 0 and no locking is active, the component can be disassembled.
To do this, go to "Extras" - "Component" and click on the Dissolve button to disassemble the component.
The component breaks down into individual items that can now be changed, removed or completely redesigned.

Bestehende Komponente ändern

Eine bestehende Komponente kann auch nachträglich verändert werden.
Hierzu muss die entsprechende Komponente aus der Galerie mit Doppelklick aufgerufen werden.
Alle Bauteile so wie auch Bearbeitungen sind frei anwählbar.
Die Komponente kann mit weiteren Elementen, Bearbeitungen, Materialien oder Beschlägen ergänzt werden.
Nach dem die Komponente abgeändert wurde,kann diese im Anschluss einfach, über das Diskettten-Symbol, gespeichert werden.
Über Speichern wird die bestehende Komponente überschrieben.
Sie muss nicht wieder als Komponente gespeichert werden.
Wenn die geänderte Komponente als neue Komponente mit neuem Namen hinzugefügt werden soll, muss diese über Komponente – Speichern unter gespeichert werden.
So lassen sich immer wiederkehrende umfangreiche Konstruktionen als variable SmartWOP Komponenten erstellen und archivieren.

Bauteilpositionierung sperren

Sie haben die Möglichkeit in einer Komponente die Bauteilpositionen zu sperren.
Das sperren der Position verschiedener Bauteile ermöglicht, dass die gesperrten Bauteile, beim verwenden der Kompoente, den gleich Abstand beibehalten, slebst wenn der Volumenkörper sich vergrößert oder verkleinert.
Wird die Komponente anschließend in ein Volumen eingezogen wird das gesperrte Maß bei der Positionierung berücksichtigt.
Alle Bauteile so wie auch Bearbeitungen sind frei anwählbar.
Die Absolute Position bezieht sich auf das Außenmaß der Konstruktion.
Das bedeutet das die Materialstärken der Bauteile mit berücksichtigt werden müssen.
Über die Pfeilrichtung der absoluten Position wird die Bezugskante bestimmt.
Mit schließen des Schlosssymbols wird der eingegebene Wert in der Komponente gesperrt.
Wird die Komponente in ein neues Volumen eingezogen, passt sich diese allseitig an.
Nur die mit dem Schlosssymbol gesperrten Bauteilen, behalten immer die vorgegebenen Positionen.

Komponente als Volumen

Wird ein Grundvolumen mit Teilern unterteilt können anschließend SmartWOP Komponenten in die einzelnen Volumen eingezogen werden.
Zum Beispiel der Aufbau einer Konstruktion mit zwei Unterschränken (Basis und Basis2) zusammen in einem Grundvolumen.
In der Stückliste werden nun alle Volumeninhalte einzeln berücksichtig.
Jede Komponente wird einzeln aufgeführt.
Über den Basis Namen werden die einzelnen Komponenten gekennzeichnet und die, in den Komponenten enthaltenen Bauteile, in der Materialliste untergeordnet.
Eingesetzte Komponenten, als Volumen, fungieren als eigenständige Konstruktionen.
Das bedeutet, dass die Inhalte der Komponenten separate Berücksichtigung finden.
Eingesetzte Komponenten, als Baugruppe hingegen, werden als ein Teil von Volumen bzw. Konstruktion gesehen.
Der Inhalt der Komponenten wird als eine Einheit berücksichtigt.

Komponente als Baugruppe

Wenn die Bauteile einer Komponente in einem Grundvolumen zusammen berücksichtigt werden sollen, können die einzelnen Komponenten als Baugruppe definiert werden.
Hierzu die Komponenten anwählen und im Eigenschaftenfenster im Untermenü Typ auf Baugruppe umstellen.
ierbei kann frei entschieden werden ob die Komponente direkt bei der Erstellung, oder bei der Verwendung, als Volumen oder Baugruppe gespeichert wird.
Eingezogene Komponenten können auch in der Konstruktion (ohne sie aufzulösen) nachträglich umgestellt werden.
In der Stückliste werden die einzelnen Komponenten (Basis und Basis2) in einem Grundvolumen nun zusammen berücksichtigt.

Komponente als Artikel

Konstruktionen die in SmartWOP als Zukaufteile benötigt werden, können als Volumentyp Artikel definiert werden.
Hierbei finden Komponenten oder Volumeninhalte keine Berücksichtigung in den Ausgaben der Stücklisten.
Artikel werden in der Beschläge Zukaufliste als Zukaufteile mit Außenmaßen ausgegeben.
In der CSV-Materialliste werden die Außenmaße unter der Option Fertigmaß, Länge, Breite und Dicke ausgegeben.
Um Zukaufteile in der CSV Liste zu berücksichtigen muss in den CSV Eigenschaften die Option Artikel aktiviert sein.
