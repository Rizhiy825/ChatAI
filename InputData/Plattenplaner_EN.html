Select and move

The button is active by default for selecting and moving workpieces.

Additive selection method

Activate the button to select multiple items.
You have the option of using the same function with a Ctrl left-click.

Subtractive selection method

Activate the button to deselect items.
You have the option of using the same function by Alt left-clicking.

Frame

Select the button to select several or all items at the same time.
Only items that are completely within the frame are selected.

Invert selection

Click the button to select all unselected items and deselect the selected ones.

Rotation

You can rotate workpieces.
Click on the arrow under the button.
Select the number of degrees by which you want to rotate the workpiece.
Activate the button and select the workpiece to be rotated.
You can rotate the volume around the desired axis by holding down the mouse button.
Clicking the button again deactivates the rotation.

Solid Mode

Select the button to switch from transparency mode to material mode.
In the submenu of the button, you have the option of switching contouring on and off in material mode.
To return to transparency mode, click on the button again.

Outer width

Specify how much the panel should be trimmed from the outer edge of the panel to the workpiece.

Inner width

Specify how much distance between the workpieces should be taken into account during nesting.

Cutting width

Enter the saw blade thickness of your machine.

Recalculate all

Click the button to recalculate the cutting plan.
All workpieces are automatically repositioned and manual optimisation is lost.

Selection to Machine

Click the button to export the selected workpieces to the machine.

All to Machine

Click the button to export all visible workpieces to the machine as a nesting program.
The function to export all items in the panel planner is only available if nesting is active.

Preview

Select a disc and click the button to display the selected board in the WOP environment.
The preview function in the panel planner is only available when nesting is active.

Area

Select the button to activate the automatic catching of workpieces.
Specify the area from which the workpieces are to be catched.

Placement

SmartWOP offers you the option of selecting the corner from which the items are to be orientated on the panel.
Click on the respective icon to recalculate the items from the selected panel corner.

Center view

Click the button to centre the view on the cutting plan.
Select a workpiece and click the button to centre the view on the selected workpiece.

Overview Panel Planner

Click on the arrow to the left of the panel name to open the cutting plan for this material.
Name/Description:SmartWOP shows you the component designations of the individual workpieces in the panel planner overview.
The panel thickness used is shown in brackets next to the material name.
Number: SmartWOP shows you the component number of the individual workpieces in the panel planner overview.
The item numbers are also displayed on the item in the panel itself.
Result Length/Width:SmartWOP shows you the finished dimensions of the individual workpieces in the panel planner overview.
Amount:SmartWOP shows you the quantity of the individual workpieces in the overview of the panel planner.
You can select and mark workpieces by clicking on them in the overview.
The panel dimensions displayed depend on the stored panel dimensions of the material.

Panel planner Template

Select a panel in the panel planner to call up the template window.
Name:SmartWOP shows you the name of the material used.
Type:SmartWOP shows you the type of material used.
Panel Name:SmartWOP shows you the name of the used board.
Length:SmartWop offers you the option of modifying the panel dimensions of the panel to be used in order to utilise leftover panels and plan the cutting in these.
Change the length of the board.
When the panel dimensions are changed, a new panel is automatically created with the entered (changed) panel dimensions.
Width:SmartWop offers you the option of modifying the panel dimensions of the panel to be used in order to utilise leftover panels and plan the cutting in these.
Change the width of the panel.
When the panel dimensions are changed, a new panel is automatically created with the entered (changed) panel dimensions.
Thickness:SmartWOP shows you the thickness of the panel used.

Panel planner display

There are various displays in the panel planner that simplify the cutting and further processing of the items.
Comments: All processing steps are displayed with a comment, numbered consecutively.
The programme name is displayed directly above the plate in the plate planner as soon as nested. Board Dimensions:SmartWOP shows you the raw dimensions of the panel used for nesting or cutting on a blue background.
SmartWOP shows you the real dimensions required for nesting or cutting on a green background.
Continuous line:The continuous line that can be found in each item marks the front edge of the item.
Dashed line:The dashed line in each item marks the lower edge of the item.
Triangle symbol with slant: The triangle symbol with the bevelled edge indicates that the item will be pushed to the workstation in the next processing step (not turned).
Triangle symbol with arc: The triangle symbol with the arc indicates that the item must be turned and placed at the workstation in the next processing step.
