Bases

Insert base

Select a base from items in lower right corner.
Pull the base with pressed left mouse button in the construction level.

Move bases

Left click the base u want to move.
Hold the left mouse button pressed on the arrow in which position you want to move the base.
Release the left mouse button when you have reached the right position.

Rotate bases

Left click the base you want to rotate.
Select the rotation button on the menu bar.
The rotate button turns blue as long you are in the rotation modus.
In the submenu of the Rotation button, select a desired number of degrees which determines how far the volume should be rotated per mouse stroke.
The arrows of the axis now turn into circles.
Hold down the left mouse button on the circle you want to rotate the base and move the mouse into this direction.
Leave the rotation mode by left clicking the rotation button again.

clone bases

Left click the volume you want to duplicate.
Select the duplicate button on the menu bar.
The volume will be duplicated with all its contents.
In the drop-down menu, you can select to mirror the selected base.

Overlapping bases

Select the Allow Overlapping button in the start bar.
Now drag the volume as described in step ,Move bases,to the desired position by using the arrows.
The allow overlapping button turns blue as long you are in the allow overlapping modus.
After switching off allow overlapping it is no longer possible to move bases into each other.

Select bases

Select the base button in the start bar.
Choose the desired base with a simple left mouse click.
If you want to select several bases at the same time, hold down the control key (CTRL) on your keyboard and select all the desired bases with a single left mouse click.

activate bases

If you selected one or several base volumes as in step "Select bases" you are able to activate the volumes, by clicking the Button "Activate volume".
SmartWOP allows you to edit only the green marked bases.
You can also select a panel and click the button "Activate volume" in this case, you activate the base in which the panel is assigned to.
This is very helpful if you are working with many separators

change base dimensions

Double left click the base in which u want to change the dimensions.
Now each side has a corresponding arrow which you can drag with the left mouse button pressed to change the dimensions of the base.
You have the possibility to use the input fields at the arrows or dimension lines to enter your desired dimensions.

Base settings

If you select a base the settings window opens.
In this window you have some setting you can change.
Name: change the name of the base

Type Select a Type for your base.
Construction kit: Select the construction kit for the volume to output the items within the volume to the superordinate volume when saving the volume as a component.
Cabinet: Select the cabinet for the base to output the items within the base to this base when saving the volume as a component.
Article: Select the article for the base in order to output the items within the base as purchased parts with an article number when saving the volume as a component.
If the volume is specified as an article, the components of the volume are not output as parts to be produced.

Width/Height/Depth change the dimensions of your base.
You can also use calculation formula.

Description You can enter a description to your selected base.
Keep in mind, that you can only read this description in this window it will not be printed on any exported list.

Count Add a number how often you want to produce this base.
Depending on how many you add the bases will appear in the material-list.

Export Reference Point: Choose between front and back.
It will place the Panel in the working progress with the front or backside to the machine stop.

Accessories Double left click on "Accessories" to open the submenu.
Single left click "Add new Accessory" to add several accessories.
Double left click "Accessory" to enter Name, supplier, article,count and description.
Accessories will appear in the Purchase List.

Transformation Double left click "Transformation" to open the submenu
Offset: You can enter the position in dimension X/Y/Z to relocate your base in the construction level.
Rotation You can enter the angle in dimension X/Y/Z to rotate your base in the construction level.

Base : Back Angled Base

In the base setting window of "Back Angled Base" you have several settings like Front Height/Back Height/Top Depth/Bottom Depth/Angle and Safety clearance to enter.
All dimension which are not registered will be calculated and added by SmartWOP automatically.
Angle The angle to be entered corresponds to the angle in the horizontal(0°-90°).
Safety clearance: Enter the safety clearance which is between the base and the cabinet.

Base : Slanted Base

In the base setting window of "Slanted Base" you have several settings like Height Left/Height Right/Width Top/Width Bottom and Angle to enter.
All dimension which are not registered will be calculated and added by SmartWOP automatically.
Angle The angle to be entered corresponds to the angle in the horizontal.
To get the angle to the right side of the base enter a negative angle.(-0° to -90°)
To get the angle to the left side of the base enter a positive angle.(0° to 90°)

Base : Corner Slanted Base

In the base setting window of "Corner Slanted Base" you have several settings like Back Width/Left Width/Front Depth and Right Depth to enter.
The angle for the opening of the Corner Slanted Base will be calculated and added by SmartWOP automatically.

Base: Roof Window

In the base setting window of "Roof Window" you have several settings to enter.
Window Width: Enter the width of the window.
Window Height: Enter the height of the window.
Window Depth: Enter the depth of the window.
Total depth: Enter the total installation depth.
Thickness: Enter the Thickness of the material.
Roof Angle: Enter the angle of the roof.
Angle Left: Enter the angle of the left panel.
Angle Right: Enter the angle of the right panel.

Setup Frame Enter the desired frame dimensions (depth/width) for the window and wall connection.
