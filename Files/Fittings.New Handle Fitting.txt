Icon:
Click on the dots in the line to open the selection menu.
Select an image stored on your computer for the icon of the fitting.
Supplier:
Indicate the supplier of the fitting.
Select an already created supplier via the drop-down selection.
Click on the X symbol in the line to delete the supplier entered in the line from your list.
Article:
Enter the item number of the handle.
Type:
Use the arrow on the right of the line to select whether the self-defined handle is a standard handle or a profile handle.
Import profile:
Define your own handle profile in the editor and import it into SmartWOP.
Import:
You have the possibility to import already defined standard handles as .VRML file.
Fittings imported as a .VRML file already contain the fitting's drilling pattern.
Positioning:
Open the drop-down menu to position the .VRML imported fittings.
Position:
Open the drop-down menu to move the imported fitting in X/Y/Z axis.
Rotation:
Open the drop-down menu to rotate the imported fitting in X/Y/Z axis.
Horizontal Distance:
SmartWOP shows you how large the horizontal distance of the handle to the outer edge of the workpiece is.
Vertical Distance:
SmartWOP shows you what the vertical distance of the handle to the outer edge of the workpiece is.
Horizontal Alignment:
SmartWOP shows you how the handle is aligned when you create it, horizontally.
Vertical Alignment:
SmartWOP shows you how the handle is aligned when you create it, in the vertical.
Absolute Position:
SmartWOP shows you the absolute position of the handle when creating the fitting.
Additional Information:
Specify additional information that you want to automatically assign to the fitting.

Switch to edit mode while creating the fitting to add or remove edits to the fitting.
When you have added the appropriate drilling pattern to the fitting, switch back to the fitting mode.
Confirm your self-defined fitting by clicking on the orange arrow in the drop-down menu of the fitting selection bar.
